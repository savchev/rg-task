# Home Assessment Task / Reward Gateway

## Setup and start

Clone the repository:

```bash
$ git clone https://github.com/savchev/rg-task.git path/to/install
```

To setup and run the project:

```bash
$ cd path/to/install
$ composer install
$ php -S 0.0.0.0:8080 -t public/ public/index.php
```

## Tests

Run PHPUnit tests:
```
$ composer test
```
